HORSE-RACING PROJECT

In this project, I explored Polish horse-racing data with Python. The data was scraped from this site. The first task was to clean the data: remove unwanted characters, change data types and formatting, split one column into two, handle missing values. Next, I answered several questions concerning the distribution of the data, trends, and performance. Each answer is accompanied by an appropriate visualisation. The libraries used: NumPy, pandas, Matplotlib.Pyplot, Seaborn.
